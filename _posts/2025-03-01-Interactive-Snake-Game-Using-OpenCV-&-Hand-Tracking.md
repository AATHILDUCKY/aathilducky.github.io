## 🚀 Interactive Snake Game Using OpenCV & Hand Tracking! 🐍🎮

Excited to share my latest project—a Snake Game controlled with Hand Tracking using OpenCV and Python! 📸✋

🔹 How it Works?
✅ Uses Index Finger Tracking for movement
✅ Built with OpenCV & MediaPipe
✅ A fun way to explore Computer Vision & AI-powered gesture control

This project showcases the power of AI, OpenCV, and real-time hand tracking in gaming! 💡 If you're interested in computer vision, AI-powered gaming, or Python projects, check it out!
